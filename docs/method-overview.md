# 大同方法论概览

## 体系结构

```
精神底色：天下为公 · 博爱
    ↓
总原则：知行合一
    ↓
┌─────────────────────────────────────────────┐
│ 第一层 认识论                                  │
│   知难行易 · 融贯中西                           │
├─────────────────────────────────────────────┤
│ 第二层 治理方法                                 │
│   三民框架 · 五权制衡 · 权能区分                  │
├─────────────────────────────────────────────┤
│ 第三层 革命战略                                 │
│   唤起民众 · 毕其功于一役 · 三阶段推进             │
├─────────────────────────────────────────────┤
│ 第四层 精神纪律                                 │
│   愈挫愈奋 · 济弱扶倾                           │
└─────────────────────────────────────────────┘
```

## 各方法速查

| 方法 | 核心问题 | 关键动作 |
|------|---------|---------|
| 知难行易 | 不确定时怎么办？ | 先行动获取信息 |
| 融贯中西 | 多个方案怎么选？ | 深入理解后创造性融合 |
| 三民框架 | 方案完整吗？ | 审查谁的、谁管、谁受益 |
| 五权制衡 | 质量怎么保障？ | 五种职能分离独立运行 |
| 权能区分 | 谁该做什么？ | 决策权归用户，执行力归 agent |
| 唤起民众 | 怎么获得支持？ | 唤起认知 + 联合平等力量 |
| 毕其功于一役 | 关联问题怎么解？ | 统一规划一次性系统解决 |
| 三阶段推进 | 变革怎么落地？ | 破旧 → 培育 → 自治 |
| 愈挫愈奋 | 失败了怎么办？ | 复盘 → 新认识 → 调整 → 重启 |
| 济弱扶倾 | 弱者被照顾了吗？ | 从最弱势视角审视方案 |

## 方法间的关系

- **互补对**：毕其功于一役 ↔ 三阶段推进（一步到位 vs 渐进推行）
- **递进链**：知难行易 → 愈挫愈奋（行动 → 失败 → 复盘 → 再行动）
- **嵌套关系**：三民框架 → 五权制衡 → 权能区分（从宏观到微观）
- **通用辅助**：济弱扶倾可在任何方法之后追加审查
